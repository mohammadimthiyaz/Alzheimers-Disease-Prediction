# Transfer Learning Framework for Prediction of Alzheimer's Disease

## Overview

This project predicts Alzheimer's Disease stages from MRI brain images using Transfer Learning and Deep Learning models.


## Research Publication

Published in IRJMETS (March 2026)

Title:
Transfer Learning Framework for Prediction of Alzheimer's Disease

DOI:
10.56726/IRJMETS91989

## Technologies Used

* Python
* TensorFlow
* Keras
* Scikit-Learn
* Google Colab

## Models Implemented

* MobileNetV2
* VGG16
* ResNet50
* EfficientNetB0

## Results

| Model                | Accuracy |
| -------------------- | -------- |
| MobileNetV2          | 68.36%   |
| EfficientNetB0       | 79.27%   |
| Tuned EfficientNetB0 | 95.14%   |
